Launch Ubantu Machine on EC2

EC2 instance has been launched and Key pair saved

this step can only be done once an EC2 instance has been launched

Console Access to the Virtual Machine

Select the virtual machine on AWS EC2 and click the “Connect” button.

Then, click on **SSH client** and note the SSH connection string provided.

Open Terminal on Ubantu virtual machine

  • open the Linux Terminal
  • Navigate to the directory where key pair file "golfblogkey1" is stored using:
cd /path/to/key-directory

Connect via SSH

Once in the correct directory, paste the SSH connection string provided:

ssh -i "yourkeyname.pem" ubuntu@ec2-12-123-1-35.ap-southwest-5.compute.amazonaws.com

yourkeyname.pem should be replaced with golfblogkey1.pem

Successful Login

there is now SSH access to the cloud machine.
