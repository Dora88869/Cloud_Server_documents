Monthly Apache & WordPress Backup Script

This script automates the monthly backup of the server.


#!/bin/bash
TIMESTAMP=$(date +"%Y-%m-%d")
BACKUP_DIR="/backup"
WEB_ROOT="/var/www/html"
DB_NAME="your_database_name"
DB_USER="your_database_user"
DB_PASSWORD="your_database_password"
ARCHIVE_NAME="wordpress_backup_$TIMESTAMP.tar.gz"

mkdir -p $BACKUP_DIR
tar -czf $BACKUP_DIR/$ARCHIVE_NAME $WEB_ROOT

mysqldump -u $DB_USER -p$DB_PASSWORD $DB_NAME > $BACKUP_DIR/wordpress_db_$TIMESTAMP.sql
tar -czf $BACKUP_DIR/wordpress_db_$TIMESTAMP.tar.gz -C $BACKUP_DIR wordpress_db_$TIMESTAMP.sql
rm $BACKUP_DIR/wordpress_db_$TIMESTAMP.sql

echo "Monthly backup completed!"
