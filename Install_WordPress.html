Install WordPress

Install WordPress on the Ubantu machine

This is step three Apache2 needs to be installed before WordPress, However this can be doen at the same time

Install Required Packages

WordPress requires **PHP** and **MySQL**, install the required packages first:

sudo apt update
sudo apt install php libapache2-mod-php php-mysql mariadb-server mariadb-client

Configure MySQL Database

  • Access MySQL:
sudo mysql -u root -p
  • Create a database and user:
CREATE DATABASE wordpress;
CREATE USER 'wpuser'@'localhost' IDENTIFIED BY 'yourpassword';
GRANT ALL PRIVILEGES ON wordpress.* TO 'wpuser'@'localhost';
FLUSH PRIVILEGES;
EXIT;

Download and Extract WordPress

Navigate to the web directory, download WordPress and extract the archive:

cd /var/www/html/
sudo wget https://wordpress.org/latest.tar.gz
sudo tar -xzvf latest.tar.gz

Configure WordPress

  • Rename the sample config file:
sudo mv wordpress/wp-config-sample.php wordpress/wp-config.php
  • Edit the config file with database details:
sudo nano wordpress/wp-config.php

Update the following lines:

define('DB_NAME', 'wordpress');
define('DB_USER', 'wpuser');
define('DB_PASSWORD', 'yourpassword');
define('DB_HOST', 'localhost');

Set Permissions and Restart Apache2

Adjust ownership and restart Apache2:

sudo chown -R www-data:www-data /var/www/html/wordpress
sudo systemctl restart apache2

Complete Installation via Web Interface

Open your browser and go to:

http://your-server-ip/wordpress

Follow the WordPress setup wizard to complete installation.
