Install Apache2

Installing Apache2 on virtual machine

This step should be done after launching an Ubantu machine in EC2

Install Apache2 on Your EC2 Instance

In the command-line access to the virtual machine, update the package repositories before installing new software as it may be out of date.

Run command:

sudo apt update

Install the Apache2 Web Server

run command to install Apache2 on EC2 instance:

sudo apt install apache2

Verify Installation

To test if Apache2 is running, open a web browser on local machine. Enter IP address the Amazon EC2 instance.
