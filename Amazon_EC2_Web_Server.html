AWS EC2 Setup Guide

Log in to the Amazon EC2 Management Console

Log into the Amazon EC2 management console: AWS EC2 Console

Launch an Ubuntu Machine in EC2

  • Launch a new instance (virtual machine).
  • Follow the steps on the AWS site.
  • Instance type: Pick a "free tier eligible" Ubuntu 20.04 Instance.
  • Add storage: Default is an 8GB hard drive
  • Add tags: No tags needed.

Configure Security Group

  • Security group name: Call it "ssh-and-web".
  • Click **Add Rule**, set traffic on ports 22 (SSH), 80 (HTTP), and 443(HTTPS).

Review and Launch

  • Click **Launch** once configurations look good.

Create a Key Pair

  • it is important that the Key Pair is saved somewhere safe.
  • it is essential for managing the virtual machine.
  • Name it **"golfblogkey"** for easy identification.

Launch Instance

  • Click **Launch Instance** to start the server.
